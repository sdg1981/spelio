# Spelio Topic Collection Vocabulary Research: Animals

This document serves as a high-quality editorial vocabulary pool of animal names for the Welsh spelling application **Spelio**. It contains common terms spanning pets, farm animals, British wildlife, birds, fish, insects, marine life, and exotic animals, sorted alphabetically by their English name.

This is a review and classification stage only. The spelling-load column judges the Welsh form first, rather than the English animal's familiarity, semantic simplicity, or frequency. Editorial status is provisional and should be reviewed before building Animals 1, Animals 2, Animals 3, or Most Common Animals.
